position: fixed; The fixed postion will stay at the same spot on your screen, regardless of scrolling, locked to the viewport and not the webpage flow. A common use of this is nagagation bars that will stay at the top of your screen, even as you read further down the page. Scroll down! position: fixed;
position: static; HTML elements are positioned static by default. Static positioned elements are not affected by the top, bottom, left, and right properties. An element with position: static; is not positioned in any special way; it is always positioned according to the normal flow of the page
left:15px;
 position:static; left:15px;
position: relative; The position: relative property lets a HTML element to be positioned relative to where it would have been and fully influenced by it's sibling elements, had it not of had a position property declared (position: static;). If no offset properties are declared, then relative positioning will have no effect.
left:15px;
 position:relative; left:15px;
position: absolute; The absolute positioning, unlike relative, will ignore it's sibling elements and instead be positioned to its closest position: absolute OR relative element. this means that it is no longer part of the document flow. By using the positioning attributes top, left, bottom and right, an element can be positioned anywhere as expected.
